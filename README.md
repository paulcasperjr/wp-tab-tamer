WP Tab Tamer
By Paul Impellizeri | Sustain Sites LLC

WP Tab Tamer is a productivity tool for WordPress developers and content creators. It solves the "Preview Loop" problem by ensuring that when you're editing, you only ever have one preview tab open at a time.

🛠️ The Problem
WordPress is notorious for "tab bloat." Every time you click Preview, a new tab opens. After an hour of editing, you end up with 10 tabs of the same post.

✅ The Solution
WP Tab Tamer is simple. It detects a new tab spawned from /wp-admin/, it checks for existing duplicates.
Keeps the Newest: The most recent preview (with your latest changes) stays open.
Tames the Rest: Older duplicate tabs are automatically closed.

Dashboard Safe: It won't close your actual WordPress Dashboard/Admin tabs.

🚀 Installation
Download: Click the green Code button and select Download ZIP. Unzip it to a folder on your machine.

Extensions Page: Open Chrome and go to chrome://extensions/.

Developer Mode: Toggle Developer mode (top right) to ON.

Load Unpacked: Click Load unpacked and select the folder containing the extension files.

Pin it: Click the puzzle icon 🧩 in your toolbar and pin WP Tab Tamer for easy access.

🛡️ Privacy & Security
Because this extension manages tabs, Chrome will display a standard warning that it can "Read your browsing history."

Sustain Sites LLC takes privacy seriously:

No Tracking: We do not track, save, or transmit any URLs or browsing data.

100% Local: All logic happens inside your browser. The extension has no "telemetry" or ability to talk to an external server.

Open Source: We encourage you to audit the background.js file to see the simplicity and security of the code.

⚙️ Usage
Automatic: Once installed, it works silently in the background whenever you are in WordPress.

Toggle Off: If you need to keep multiple duplicates open for a specific task, right-click the extension icon and select Disable WP Tab Tamer.
